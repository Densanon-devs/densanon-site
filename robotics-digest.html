Research & Papers

Disentangled Robot Learning via Separate Forward and Inverse Dynamics Pretraining

ArXiv cs.RO
  • Identifies a core tension in VLA models: 2D image forecasting objectives and 3D action prediction objectives are fundamentally misaligned when trained jointly
  • Proposes separate pretraining of forward dynamics (what the world will look like) and inverse dynamics (what action causes a transition) as a disentanglement fix
  • Decoupled training improves generalization and sample efficiency compared to end-to-end VLA baselines — relevant for anyone building generalist robot policies

BrainMem: Brain-Inspired Evolving Memory for Embodied Agent Task Planning

ArXiv cs.RO
  • Addresses long-horizon task planning for embodied agents in 3D environments, where success depends on both immediate perception and accumulated cross-task experience
  • Draws inspiration from human memory consolidation — separating episodic (recent) from semantic (distilled long-term) memory in the agent's planning loop
  • LLM-based planners typically have no persistent memory across tasks; BrainMem's evolving memory architecture is a direct answer to that gap

Visual-RRT: Finding Paths toward Visual-Goals via Differentiable Rendering

ArXiv cs.RO
  • Standard RRT motion planners require goals specified as explicit joint angles — Visual-RRT instead accepts an image of the desired end state as the goal
  • Uses differentiable rendering to bridge the gap between visual goal specification and the joint-space search tree
  • Enables a more intuitive 'show the robot what you want' interface for manipulation planning without pre-specified numerical configurations

Improving mobile robot stability on uneven terrain through vibration monitoring techniques

The Robot Report
  • Vibration sensors (IMUs and accelerometers) can serve as a cheap, real-time proxy for terrain stability — detecting instability before it causes a tip or navigation failure
  • The technique is especially relevant for field robots operating in agriculture, construction, and outdoor inspection where terrain is unpredictable
  • Adds a lightweight, hardware-accessible safety layer that doesn't require expensive terrain mapping or deep learning
