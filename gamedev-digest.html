Building Mesh-Free Portal Bubbles in UE5 with Post-Process

80 Level
  • Uses UE5's post-process pipeline combined with SceneCapture2D to render portal views without requiring a dedicated mesh object in the scene
  • Key challenge solved: frame synchronization between the gameplay camera and SceneCapture to avoid temporal artifacts in the captured portal image
  • Purely a VFX/shader approach — no custom C++ required, making it accessible to blueprint-heavy projects
  • Useful pattern for sci-fi or fantasy games needing stylized 'window to another world' effects without complex scene management

Letting AI play my game – building an agentic test harness to help play-testing

Hacker News Game Dev
  • Developer built a working agentic harness that lets an AI model autonomously play through a game for automated playtesting — 27 upvotes on HN at time of fetch
  • Agent observes game state via structured output, makes decisions each tick, and logs outcomes — enabling regression testing of game feel and balance without human playtesters
  • Key insight: framing the game's state as an LLM-readable context (not raw pixels) is what makes the approach tractable with current models
  • Practical alternative to expensive human QA sessions for solo devs and small studios; the harness pattern is portable to most turn-based or state-machine games
