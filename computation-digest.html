New 'GeForge' and 'GDDRHammer' attacks can fully infiltrate your system through Nvidia's GPU memory

Tom's Hardware
  • GeForge and GDDRHammer are rowhammer-style attacks that exploit bit-flip vulnerabilities in GDDR6/6X VRAM, allowing attackers to escalate privileges or escape sandboxed environments via the GPU.
  • Unlike DRAM rowhammer mitigations, VRAM lacks equivalent ECC protections in consumer cards, making the attack surface difficult to patch without hardware-level changes.
  • The research has immediate implications for cloud GPU tenants sharing physical hardware — multi-tenant GPU environments may need isolation guarantees analogous to those already applied to CPU memory.

AWS Engineer Reports PostgreSQL Performance Halved By Linux 7.0, But A Fix May Not Be Easy

Phoronix
  • An AWS engineer documented a ~50% throughput regression in PostgreSQL workloads after upgrading to Linux 7.0, traced to changes in memory management and scheduler behavior introduced in the new kernel.
  • The regression is particularly sharp on high-connection OLTP workloads, the exact profile of most cloud database deployments — making this a critical issue for managed PostgreSQL services.
  • Kernel maintainers have acknowledged the report, but the interaction between PostgreSQL's locking model and the new kernel paths is complex enough that a clean fix may require coordination across multiple subsystems.

3mdeb Makes Progress Bringing AMD openSIL + Coreboot To Ryzen AM5 Motherboard

Phoronix
  • 3mdeb has achieved early boot milestones using AMD's open-source openSIL firmware initialization library paired with coreboot on an AM5 platform — a meaningful step toward fully open firmware on modern AMD desktop hardware.
