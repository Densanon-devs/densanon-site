AMD FP-DSS Security Bug For Zen 1 CPUs Made Public, Linux Kernel Patched

Phoronix
  • Floating Point Divider State Sampling (FP-DSS) is a newly disclosed side-channel vulnerability affecting original Zen 1 and Zen 1+ processors.
  • Exploiting it could allow cross-process leakage of floating-point division state — a speculative execution class issue.
  • Linux kernel patch is already merged; users on first-gen Ryzen or EPYC hardware should update promptly.
