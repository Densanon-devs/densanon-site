Hardware & Chips

TSMC SoIC 3D stacking roadmap outlines path from 6-micron pitches today to 4.5-micron in 2029 — Fujitsu's Monaka CPU to benefit from face-to-face chiplet stacking

Tom's Hardware
  • Current SoIC production runs at 6 µm bump pitch; the roadmap adds a 6.5 µm variant for yield, then targets 4.5 µm by 2029
  • Face-to-face (F2F) stacking mode is newly added alongside existing face-to-back — enables tighter signal coupling between logic dies without a passive interposer
  • Fujitsu's Monaka HPC CPU is a named production customer for F2F SoIC, confirming the technology is committed rather than roadmap vapor
  • Tighter pitch reduces interconnect length and latency between stacked chiplets — directly impactful for memory-bandwidth-bound HPC and AI workloads

US stops exports of tools to China's number two chip maker — Hua Hong and Huali Microelectronics reportedly on the cusp of starting a 7-nm fab in Shanghai

Tom's Hardware

More details emerge about how Intel now earns more revenue from each wafer by looking to the edges — analyst reports say reduced yield variability across each wafer leads to more sellable CPUs

Tom's Hardware
  • Intel is tightening process controls at wafer edges — historically a low-yield dead-zone — to recover more sellable dies per wafer
  • Analyst reports attribute this to incremental fab optimization rather than new node introductions, meaning no additional capex required
  • More sellable dies per wafer improves gross margin directly; under AI-driven demand, squeezing output from existing Intel 3/18A capacity has outsized revenue impact
  • Technique addresses a fundamental fab challenge: process parameters drift toward wafer edges, making edge dies historically unreliable for binning to high-performance SKUs

Hygon C86-4G CPU Support Added To The GCC 17 Compiler

Phoronix
  • GCC 17 gains support for three Hygon C86-4G server variants: C86-4G-M4, C86-4G-M6, C86-4G-M7 — all x86_64 processors targeted at Chinese data centers
  • Merged to GCC mainline (not a fork), so standard Linux toolchains will natively emit optimized code for this silicon
  • Hygon is architecturally descended from AMD Zen 1 IP licensed before export controls tightened; the C86-4G line represents independent iteration on that base
  • GCC 16.1 is imminent; this targets GCC 17, meaning production compiler support is approximately one year out

Tom's Hardware
