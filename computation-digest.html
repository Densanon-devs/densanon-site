Intel VP claims up to 30% of CPU performance is untapped by modern games

Tom's Hardware
  • Intel argues hybrid P-core/E-core architectures are underutilized because game engines weren't written for heterogeneous thread scheduling.
  • Up to 30% additional throughput is theoretically available without any hardware change — purely through better thread affinity and task prioritization.
  • Intel is pushing developer tooling and scheduler-aware APIs to help titles actually use the E-core cluster, which sits mostly idle today.

AMD EXPO 1.2 arrives with three Chinese memory vendors — gains may be muted until Zen 6

Tom's Hardware
  • EXPO 1.2 adds certified support for memory from three Chinese vendors, expanding the ecosystem beyond the Samsung/Hynix/Micron triopoly.
  • The new spec enables higher DDR5 speed bins and tighter timings on AM5 motherboards via BIOS updates.
  • Real-world performance gains will be modest on current Zen 5 hardware — the full benefit is gated on Zen 6's improved memory controller.

Microsoft Reportedly Looking At Rebasing Azure Linux On Fedora

Phoronix
  • Azure Linux (used inside Azure cloud, WSL, and other Microsoft infrastructure) is reportedly being considered for a full rebase onto Fedora's package base.
  • Currently Azure Linux is a custom minimal distro maintained internally by Microsoft — a Fedora foundation would give it access to a much larger community-maintained package ecosystem.
  • Fedora's close relationship with RHEL means enterprise tooling and certifications would follow more naturally.

Coinbase Advisers Warn Quantum Computing Will Crack Blockchain Encryption — Window to Prepare Is Narrowing

The Quantum Insider
  • Coinbase's advisory board published a position paper stating that a quantum computer capable of breaking Bitcoin and Ethereum's elliptic curve cryptography will eventually be built.
  • The paper stresses timeline urgency: post-quantum migration for blockchain is complex and slow, so preparation must start well before hardware reaches that threshold.
  • NIST's recently finalized post-quantum cryptographic standards are the migration target — but retrofitting blockchain consensus and wallet infrastructure is a major engineering effort.
  • The paper frames the preparation window as narrowing faster than the industry is currently moving.

Welinq: Using Shared Entanglement to Network Quantum Processors Into Modular Scale-Out Clusters

The Quantum Insider
  • Welinq is building quantum networking infrastructure that uses shared entanglement to interconnect separate quantum processors into modular clusters.
  • The approach enables scale-out to higher qubit counts without requiring a single monolithic processor — a key path to commercially relevant quantum systems.
  • Near-term commercial product is quantum-safe connectivity between data centers, separate from the full quantum computing use case.
  • Technical building blocks include qubit-photon interfaces, entanglement distribution, and repeater architecture for longer-distance quantum links.
