Hardware & Chips

NEO Semiconductor's 3D X-DRAM for AI Processors Passes Proof-of-Concept Validation

Tom's Hardware
  • NEO Semiconductor's 3D X-DRAM uses existing 3D NAND fabrication processes to build a cheaper alternative to HBM for AI accelerators.
  • Proof-of-concept validation has been completed and the company has secured funding to advance to next-gen development.
  • If commercialized, this could undercut HBM on cost while targeting similar bandwidth-hungry AI inference workloads.
  • 3D NAND process reuse is the key bet here — it means NEO doesn't need entirely new fab infrastructure, lowering the barrier to production.

CPU Requirements for AI Workloads Are Multiplying, Driving Intensifying Shortages and Price Hikes

Tom's Hardware
  • Inference and agentic AI workloads are pushing CPU-to-GPU ratios toward 1:1 in server deployments, reversing the GPU-heavy trend.
  • Intel is already shifting production capacity from consumer chips toward Xeon to meet surging server CPU demand.
  • Agentic pipelines require sustained CPU throughput for orchestration, memory management, and I/O — tasks GPUs don't handle efficiently.

Farewell ISDN, Ham Radio & Old Network Drivers: Linus Torvalds Merges 138k Lines Removed

Phoronix
  • Linus merged a pull request removing 138,000 lines of code — the entire ISDN subsystem plus numerous legacy PCMCIA-era network drivers.
  • The removals were partly motivated by AI/LLM-generated bug reports flooding upstream maintainers for code with zero real users.
  • Less dead code means smaller attack surface, faster build times, and less noise in CI — a genuine maintenance win for the kernel.

HDMI FRL Support Achieved with Open-Source Nouveau for NVIDIA GPUs

Phoronix
  • The open-source Nouveau driver for NVIDIA GPUs now supports HDMI Fixed Rate Link (FRL), a core feature of the HDMI 2.1 specification.
  • This contrasts with AMD's AMDGPU driver, which remains blocked from HDMI 2.1 implementation by HDMI Forum licensing restrictions on open-source.
  • FRL support enables higher resolutions and refresh rates over HDMI, making Nouveau more viable as a daily-driver open-source GPU stack.
