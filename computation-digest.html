RADV Driver Enables Host Image Copy By Default For RDNA2 & Newer

Phoronix
  • VK_EXT_host_image_copy is now enabled by default in the RADV Mesa driver for RDNA2 and newer GPUs, eliminating the need to stage asset data through CPU-accessible buffers during loads.
  • The change reduces memory usage and latency during texture/asset loads by allowing direct CPU-to-GPU image data transfers on the host processor.
  • This Vulkan 1.3.258 extension has been available since 2023 but is only now on by default in RADV, bringing a concrete throughput improvement to Linux AMD GPU users.

Linux 7.1 Lands Workaround For Arm C1-Pro Erratum

Phoronix
  • Linux 7.1 includes a kernel-level workaround for a confirmed hardware bug in ARM's C1-Pro CPU affecting its Scalable Matrix Extension (SME) implementation.
  • The erratum required a kernel mitigation rather than a firmware/microcode patch, making upstream kernel tracking the fix path for affected datacenter and HPC deployments.
  • ARM C1-Pro is a high-performance server-class core, so this fix is relevant for any Linux ARM infrastructure running SME workloads.

Initial Linux Driver Patches For Smart Data Accelerator Interface "SDXI"

Phoronix
  • Initial kernel patches for SDXI (Smart Data Accelerator Interface) have been posted — a vendor-neutral architecture for offloading memory-to-memory data movement from the CPU.
  • SDXI is designed to let hardware accelerators handle bulk data copies, reducing CPU overhead during data-intensive network, storage, and AI pipeline transfers.
  • Being vendor-neutral means the driver is intended to work across multiple hardware implementations rather than being tied to a single vendor's DMA engine.

Scientists solve decades-old 2D physics puzzle — Chaotic growth in a 2D quantum system obeys statistical laws

Tom's Hardware
  • Physicists confirmed for the first time that chaotic surface growth in a 2D quantum system follows the Kardar-Parisi-Zhang (KPZ) universality class, validating a 40-year-old mathematical model.
  • The KPZ framework describes how random, uneven surfaces evolve statistically over time — confirming it holds in quantum systems is a foundational result for quantum chaos research.
  • This finding has implications for understanding noise, decoherence, and error propagation in 2D quantum computing architectures.
