AMD Posts Newest Linux Patches To Accelerate Page Migration For Better Performance

Phoronix
  • AMD engineers published v4 of the accelerated page migration patch series, originally started by NVIDIA in early 2025, now extended with batch copy support and hardware offload paths.
  • The patches target NUMA-heavy workloads and GPU-adjacent memory movement, where page migration latency is a real bottleneck for HPC and ML inference pipelines.

OpenCL Introducing Cooperative Matrix Extensions For Machine Learning

Phoronix
  • The OpenCL API is adopting Cooperative Matrix extensions, mirroring the Vulkan cooperative matrix support introduced in 2023 and the SPIR-V integration that followed it.
  • Cooperative matrix ops allow workgroups to collaborate on matrix multiply-accumulate at the hardware level — the same primitive underlying tensor core performance on modern GPUs.
  • Bringing this to OpenCL expands native ML acceleration to a broader set of GPU hardware and runtimes beyond the Vulkan and CUDA ecosystems, relevant for cross-vendor HPC deployments.

CPPC v4 Support Being Worked On By NVIDIA For The Linux ACPI Driver

Phoronix
  • NVIDIA is implementing CPPC v4 (Collaborative Processor Performance Control) in the Linux ACPI driver, following the ACPI 6.6 specification published last year.
  • CPPC provides a standardized OS-level interface for fine-grained CPU core performance management using an abstract scale — v4 extends what the OS scheduler and power management stack can request from hardware.
  • NVIDIA's involvement signals this will land on GPU-adjacent server platforms where tightly coupled CPU core tuning alongside GPU workloads matters for total system power efficiency.
