Linux 7.1 Expected To Begin Removing i486 CPU Support

Phoronix
  • A patch queued for the Linux 7.1 merge window starts the formal removal of Intel 486 support — Linus Torvalds confirmed there is 'zero real reason' to continue maintaining it.
  • The i486 launched in 1989 and lacks key instructions modern Linux relies on, meaning the support layer adds complexity with no practical beneficiaries running upstream kernels.
  • Removing legacy ISA targets allows the kernel to simplify low-level boot paths and memory management code that currently carries i486-specific workarounds.

Google Proposes JSIR As A High-Level IR For JavaScript

Phoronix
  • Google is proposing JSIR — a high-level intermediate representation for JavaScript — already used internally for code analysis, bytecode-to-JS transpilation, and deobfuscation at production scale.
  • Unlike low-level IRs such as LLVM bitcode, JSIR preserves JavaScript semantics, making it far better suited for security auditing, reverse engineering, and static analysis of JS bundles.
  • A standardized JS IR would benefit the broader tooling ecosystem: bundlers, linters, sanitizers, and AI code analysis tools could all share a common structured representation rather than operating on raw ASTs.
